The GenSing Collaborative Grapher is the offline version of the CN-ECT Collaborative Grapher. It has the same interface and functinalities as the CN-ECT Collaborative Grapher except for database connectivity. Student contributions are saved into a spreadsheet with a .csv extension. Geogebra galleris ( if any were saved during the activity ) will be have the extension of .ggb.

For detailed documentation on the model's interface, please see the file index.html in the folder GCG_Help
